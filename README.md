# 🍦 Neo's Ice Cream Parlor

A responsive web design mockup created for a **VGen HTML/CSS Challenge**.

The objective of this project was to design a themed website centered around **ice cream**, using a cohesive color palette, clean layout, and responsive design principles. The design takes inspiration from **Neapolitan ice cream**, combining strawberry, vanilla, and chocolate colors throughout the interface while incorporating subtle RWBY-inspired references.

## ✨ Features

- Responsive mobile-first layout
- Neapolitan-inspired color palette
  - 🍓 Strawberry header
  - 🍦 Vanilla background
  - 🍫 Chocolate pricing table and footer
- Ice cream flavor menu
- Pricing table with scoop sizes
- Rounded UI components
- Simple, clean typography
- RWBY-inspired references (Lien & Remnant)
